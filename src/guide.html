Creating Account

So you need to create an account. Here are some handy tips to make sure that everything gets filled out correctly:

  1. Make sure everything is filled out with the proper information (numbers with numbers, etc)!
  2. Make sure that the starting balance has numbers in it, even if it is 0.0
  3. Description is yours to do with you will, but I would recommend something that will remind you what this person is. (i.e. professor, student, etc).
  4. Phone and email should be pretty self explanatory; you might find it helpful to use their University information, but fill in what you think is necessary.

Making Transaction
